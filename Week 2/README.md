# Student Grade Calculator

## Overview

This project is a Python-based Student Grade Calculator developed as part of Week 2 Internship Tasks.

The program takes a student's name and marks as input, validates the marks, calculates the grade, and displays an encouraging message based on performance.

## Features

* Accepts student name and marks
* Input validation for marks (0-100 only)
* Handles invalid inputs using error handling
* Uses if-elif-else statements for grading
* Uses a while loop for validation
* Uses functions for reusable code
* Displays encouraging messages

## Grading System

| Marks  | Grade |
| ------ | ----- |
| 90-100 | A     |
| 80-89  | B     |
| 70-79  | C     |
| 60-69  | D     |
| 0-59   | F     |

## Technologies Used

* Python 3

## How to Run

1. Open terminal.
2. Navigate to the project folder.
3. Run:

python3 grade_calculator.py
